En esta rama se sigue lo establecido en la rama main, solo se implementa el middleware para que pueda validar la existencia de la peticion en la cache de manera separada y mantener un controller mas limpio

## Middleware
Es una capa de filtracion, se revisa antes de los FormRequest. Tiene como funcion inspeccionar, modificar o rechazar la peticion.
